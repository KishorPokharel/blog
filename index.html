Makefile

Sun, Aug 8, 2021 What is a makefile A Makefile is a text file that defines a set of rules to be executed. Then with the help of make utility you can execute the target which runs all command inside that rule. If you want to follow along, make sure you have the make utility installed on your machine. Basic syntax Here is what the syntax looks like target: prerequisites <TAB>command The prerequisites can be other targets and they are optional.
