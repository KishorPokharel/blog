Sun, Aug 8, 2021

What is a makefile

A Makefile is a text file that defines a set of rules to be executed. Then with the help of make utility you can execute the target which runs all command inside that rule. If you want to follow along, make sure you have the make utility installed on your machine.

Basic syntax

Here is what the syntax looks like

target: prerequisites
<TAB>command

The prerequisites can be other targets and they are optional.

Hello World

Now that you have the basic syntax let’s print hello world using Makefile

Create a file Makefile

$ touch Makefile

Add the following lines to the file

hello:
	echo "Hello World"

Now run make in the terminal. Make sure you’re in the right folder. You should see something like this as output.

$ make
echo "Hello World"
Hello World

Notice that the make echoes the actual command.

Supress echoing of actual command

We don’t want the actual echo command to be printed. To supress echoing the actual command we can use @ in front of command like this

hello:
   @echo "Hello World"

Now if you run make again you can see the following output.

$ make
Hello World

Comments on Makefile

You can use # to create comment

#this target says hello
hello:
	@echo "Hello World"

Variables and Environment Variables

You can access environment variables using ${ENV} this syntax.

Shell Commands

You can execute shell commands inside of a command in a Makefile.
